# AP Classroom Answer Hider

A Chrome extension that hides correct answers and explanations on AP Classroom while studying, helping you focus on learning without spoilers.

## Features

- Automatically hides answer indicators (correct/incorrect colors) on AP Classroom pages
- Floating toggle panel to show/hide answers on demand
- Collapsible panel to minimize when not needed
- Works on all AP Classroom pages

## Installation

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the folder containing this extension
5. The extension will be installed and active

## Usage

1. Navigate to any AP Classroom page
2. You'll see a floating panel in the bottom-right corner labeled "Answer Hider"
3. The toggle switch controls whether answers are hidden (default: on)
4. The status indicator shows "Hidden" or "Showing"
5. Click the "-" button to collapse the panel, "+" to expand it

## How it Works

The extension injects CSS styles that hide visual indicators of correct/incorrect answers and explanations. The toggle allows you to reveal answers when you need to check your work or study the solutions.

## Version

Current version: 1.0

## Compatibility

- Chrome (and Chromium-based browsers)
- AP Classroom website

## Privacy

This extension only runs on AP Classroom pages and does not collect or transmit any data.
